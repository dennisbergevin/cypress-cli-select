> npx cypress-cli-select run --help

Interactive cli prompts to select Cypress specs, tests or tags run

npx cypress-cli-select run [args]

Options:
  --version              Show version number                           [boolean]
  --print-selected       Prints selected specs, tests or tags          [boolean]
  --choose-spec-pattern  Uses specPattern to enable spec ordering      [boolean]
  --help                 Show help                                     [boolean]

Examples:
  npx cypress-cli-select run --print-selected
  npx cypress-cli-select run --component --choose-spec-pattern
  npx cypress-cli-select run --browser=firefox
